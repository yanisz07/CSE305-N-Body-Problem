# N-Body Simulation

**Prerequisite:** Magick++ must be installed.

1. Open a terminal and `cd` into the project folder (where `n_body_basic.cpp` lives).

2. To Compile:
g++ -std=c++17 -O2 n_body_basic.cpp $(Magick++-config --cxxflags --libs) -o n_body_basic.exe


3. To Run:
./n_body_basic.exe
